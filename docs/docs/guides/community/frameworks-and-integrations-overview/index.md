# Frameworks and Integrations Overview

Using OpenRouter with Popular Frameworks and Integrations

OpenRouter integrates seamlessly with popular AI frameworks and SDKs. Choose your preferred framework below for detailed integration guides:

## Available Framework Integrations

- **[Effect AI SDK](../../../community/effect-ai-sdk/index.md)** - Integration with TypeScript Effect applications using the Effect AI SDK
- **[LangChain](../../../community/lang-chain/index.md)** - Integration with LangChain for Python and JavaScript applications
- **[LlamaIndex](https://developers.llamaindex.ai/python/framework-api-reference/llms/openrouter)** - Integration with LlamaIndex for Python and TypeScript RAG applications
- **[Mastra](../../../community/mastra/index.md)** - Unified interface for AI model access through Mastra framework
- **[OpenAI SDK](../../../community/open-ai-sdk/index.md)** - Direct integration using the official OpenAI SDK for Python and TypeScript
- **[PydanticAI](../../../community/pydantic-ai/index.md)** - High-level interface for Python applications using PydanticAI
- **[TanStack AI](../../../community/tanstack-ai/index.md)** - Integration with TanStack AI for React, Solid, and Preact applications
- **[Vercel AI SDK](../../../community/vercel-ai-sdk/index.md)** - Integration with Next.js applications using the Vercel AI SDK

## Other Integrations:

- **[Aider](https://aider.chat/docs/llms/openrouter.html)** - Integration with Aider coding assistant
- **[Cline](https://docs.cline.bot/provider-config/openrouter)** - Integration with Cline coding assistant
- **[Deep Agents CLI](https://docs.langchain.com/oss/python/deepagents/cli/providers)** - Integration with Deep Agents terminal coding agent
- **[Junie CLI](../../../community/junie/index.md)** - Integration with JetBrains Junie agentic coding tool
- **[Kilo Code](https://kilocode.ai/docs/providers/openrouter)** - Integration with KiloCode coding assistant
- **[Langfuse](../../../community/langfuse/index.md)** - Integration with Langfuse Observability and Tracing
- **[Roo Code](https://docs.roocode.com/providers/openrouter)** - Integration with Roo Code coding assistant
- **[VSCode Copilot](https://code.visualstudio.com/docs/copilot/customization/language-models)** - Integration with VSCode Copilot
- **[Xcode](../../../community/xcode/index.md)** - Integration with Xcode coding assistant

You can also find additional examples in our [GitHub repository](https://github.com/OpenRouterTeam/openrouter-examples).
